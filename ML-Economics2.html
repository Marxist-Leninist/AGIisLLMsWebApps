Python Simulation of Agents Influenced by Maslow's Hierarchy of Needs

Introduction

This simulation models an economy with agents whose behaviors are influenced by Maslow's Hierarchy of Needs. Agents prioritize their actions based on their current level of needs, starting from physiological needs up to self-actualization.

Python Code


import random

# Define Maslow's Hierarchy of Needs levels
class NeedsLevel:
    PHYSIOLOGICAL = 1
    SAFETY = 2
    LOVE_BELONGING = 3
    ESTEEM = 4
    SELF_ACTUALIZATION = 5

# Base Agent class
class Agent:
    def __init__(self, agent_id, role, wealth, energy, land):
        self.agent_id = agent_id
        self.role = role
        self.wealth = wealth
        self.energy = energy
        self.land = land
        self.current_need = NeedsLevel.PHYSIOLOGICAL
        self.alive = True

    def update_needs(self):
        # Simplified logic to update needs based on certain thresholds
        if self.wealth < 20:
            self.current_need = NeedsLevel.PHYSIOLOGICAL
        elif self.wealth < 50:
            self.current_need = NeedsLevel.SAFETY
        elif self.wealth < 100:
            self.current_need = NeedsLevel.LOVE_BELONGING
        elif self.wealth < 150:
            self.current_need = NeedsLevel.ESTEEM
        else:
            self.current_need = NeedsLevel.SELF_ACTUALIZATION

    def act(self, market):
        if not self.alive:
            return

        self.update_needs()

        if self.current_need == NeedsLevel.PHYSIOLOGICAL:
            self.acquire_basic_needs(market)
        elif self.current_need == NeedsLevel.SAFETY:
            self.ensure_safety(market)
        elif self.current_need == NeedsLevel.LOVE_BELONGING:
            self.social_interactions(market)
        elif self.current_need == NeedsLevel.ESTEEM:
            self.build_esteem(market)
        elif self.current_need == NeedsLevel.SELF_ACTUALIZATION:
            self.self_actualize(market)

        self.check_survival()

    def acquire_basic_needs(self, market):
        # Purchase food and water if affordable
        if self.wealth >= market.food_price:
            self.wealth -= market.food_price
            market.sell_food()
            print(f"Agent {self.agent_id} bought food for {market.food_price} units.")
        else:
            print(f"Agent {self.agent_id} could not afford food.")

    def ensure_safety(self, market):
        # Purchase housing or save money
        if self.wealth >= market.housing_price:
            self.wealth -= market.housing_price
            market.sell_housing()
            print(f"Agent {self.agent_id} bought housing for {market.housing_price} units.")
        else:
            print(f"Agent {self.agent_id} is saving for housing.")

    def social_interactions(self, market):
        # Interact with other agents
        print(f"Agent {self.agent_id} is socializing.")

    def build_esteem(self, market):
        # Purchase luxury goods
        if self.wealth >= market.luxury_price:
            self.wealth -= market.luxury_price
            market.sell_luxury()
            print(f"Agent {self.agent_id} bought luxury goods for {market.luxury_price} units.")
        else:
            print(f"Agent {self.agent_id} could not afford luxury goods.")

    def self_actualize(self, market):
        # Engage in self-improvement activities
        print(f"Agent {self.agent_id} is pursuing self-actualization.")

    def check_survival(self):
        # Agents die if they cannot meet physiological needs
        if self.current_need == NeedsLevel.PHYSIOLOGICAL and self.wealth < 1:
            self.alive = False
            print(f"Agent {self.agent_id} has died due to unmet physiological needs.")

# Market class
class Market:
    def __init__(self):
        self.food_stock = 1000
        self.food_price = 5
        self.housing_stock = 500
        self.housing_price = 50
        self.luxury_stock = 200
        self.luxury_price = 100

    def sell_food(self):
        if self.food_stock > 0:
            self.food_stock -= 1

    def sell_housing(self):
        if self.housing_stock > 0:
            self.housing_stock -= 1

    def sell_luxury(self):
        if self.luxury_stock > 0:
            self.luxury_stock -= 1

# Initialize agents
agents = []
for i in range(1, 151):
    role = random.choice(['Worker', 'Capitalist', 'Farmer', 'Technologist', 'Consumer'])
    wealth = random.uniform(20, 200)
    energy = 50
    land = 10 if role == 'Farmer' else 0
    agents.append(Agent(agent_id=i, role=role, wealth=wealth, energy=energy, land=land))

# Initialize market
market = Market()

# Simulation rounds
for round_number in range(1, 11):
    print(f"\n--- Round {round_number} ---")
    for agent in agents:
        agent.act(market)

    # Check if market resources are depleted
    if market.food_stock <= 0:
        print("Food stock depleted. Simulation ends.")
        break

# Final status
print("\nFinal Agent Status")
print("Agent ID\tRole\t\tWealth\t\tAlive")
for agent in agents:
    status = "Alive" if agent.alive else "Deceased"
    print(f"{agent.agent_id}\t\t{agent.role}\t{agent.wealth:.2f}\t\t{status}")

Explanation of the Code

Maslow's Hierarchy of Needs

Implemented as a class NeedsLevel with levels from physiological needs to self-actualization. Each agent has a current_need attribute that represents their current level in Maslow's hierarchy. Agents update their needs based on their wealth, simulating the idea that as basic needs are met, they move up the hierarchy.

Agent Actions

  • Physiological Needs: Agents attempt to purchase food. If they cannot afford it, they risk dying.
  • Safety Needs: Agents try to purchase housing. If unaffordable, they save money.
  • Love and Belonging: Agents engage in social interactions (simulated with a print statement).
  • Esteem Needs: Agents purchase luxury goods if they can afford them.
  • Self-Actualization: Agents pursue self-improvement activities (simulated with a print statement).

Market Class

The market has stocks and prices for food, housing, and luxury goods. Methods are provided to sell goods to agents, reducing stock accordingly.

Simulation Loop

The simulation runs for a set number of rounds (e.g., 10). In each round, every agent acts based on their current needs level. The simulation checks for resource depletion after each round.

Agent Survival

Agents die if they cannot meet their physiological needs and have insufficient wealth. The check_survival method handles this logic.

Final Status Output

After the simulation, the final status of each agent is printed, showing their ID, role, wealth, and whether they are alive.

Notes and Considerations

  • Social interactions and self-actualization are represented by print statements for simplicity.
